# ens cafe

> the community marketplace for ENS names

[ens cafe](https://ens.cafe) is a marketplace built specifically for [ENS](https://ens.domains) names.

## getting started

```
yarn install
yarn dev
```

## tech stack
* Next.js
* React
* Tailwind CSS
* █████████ Protocol
